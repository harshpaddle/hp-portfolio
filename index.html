OverWatch WorkShop

Summary

OverWatch WorkShop is an online social platform for overwatch gamers to view what's trending in its community. Once logged in using google, user has the ability to create post with usually a media file if they choose, delte post(if owner), comment on post, upvote(like) and downvote(dislike).

Motive
The idea was brought to life after one of the team members, of this project, expressed his frustration keeping up with online events, trends and updates continuosly coming to the much beloved OverWatch game. The subReddit page created for the purpose of doing these things was polluted with useless content/spam and just a mess. It was then We then decided to create a subReddit clone with same functions such as creating user accounts, posting content with scores (upvotes- downvotes) and string of comments per post. These functionalities require strong back-end application with secure server, which was built using React.js, Passport.js authentication with Google Oauth, strong and secure Sequelize database, among other technologies.

Crystal Game

Summary

Bag Them Crystals is a simple game where at the start of each round you're given an 'empty bag' and an exact weight(number/value) that is to be achieved adding one of three crystals given, each assigned random weight at the beginning of every game, to your bag.

The game is won when the weight of your bag matches perfectly with the given weight and lost if your bag's weight goes over the exact weight to be matched.

You're supposed to figure out the weight(value) of each crystal by addding them to your bag and noting the difference it make in total weight. You then keep adding different crystals to your bag moving closer to the exact weight to match, making sure not to jump over it.

Motive
This project was sort of a prework assignment, required to be done before attending the Rutgers Coding Bootcamp. I believe it was assigned to sharpen student's problem-solving/self-dependency skills as it made you use lot of outside resources and documentations.
