Ticketify

OverWatch WorkShop is an online social platform for overwatch gamers to view what's trending in its community. Once logged in using google, user has the ability to create post with usually a media file if they choose, delte post(if owner), comment on post, upvote(like) and downvote(dislike).

Motive
The idea was brought to life after one of the team members, of this project, expressed his frustration keeping up with online events, trends and updates continuosly coming to the much beloved OverWatch game. The subReddit page created for the purpose of doing these things was polluted with useless content/spam and just a mess. It was then We then decided to create a subReddit clone with same functions such as creating user accounts, posting content with scores (upvotes- downvotes) and string of comments per post. These functionalities require strong back-end engine which was built using React.js, Passport.js authentication with Google Oauth, strong and secure Sequelize database, among other technologies.

Powered by:

Ticketify

Ticketify is an application created for music listeners that lets users import thier music playlists from their spotify account and list live events and concerts with an option to purchase ticket among other information about music artists in their playlist. Application uses two APIs to function, one from spotify which allows user to log in and retrieve their music library and playlists, and second that looks up each artist from selelcted user playlist coming near tri-state area using ticketmaster.
